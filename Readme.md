# Desktop Utility that shows Tasks assigned to you:

Combines:

- Jira - Tickets
- GitHub - PRs & Issues
- GitLab - MRs
- Bitbucket - PRs
- Azure DevOps - PRs

Into a unified view.

## Links

- [Website](https://pr.bfanger.nl/)

## Build desktop app

```sh
cd electron
npm install
npm run build
open out/
```
